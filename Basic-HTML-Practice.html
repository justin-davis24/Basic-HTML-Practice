What is Web Application Development About?

Web Application is about making websites using different types of coding or programming languages.

A couple of examples would be:

  • C++
  • JavaScript
  • Python

    • Another example would be something like:

      HTML

    The difference between the first few and HTML is, C++, JavaScript, and Python are coding languages whereas HTML is just a programming language
